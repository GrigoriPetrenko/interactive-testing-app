Load Questions from File

Please select a JSON file containing your test questions.
File Format Requirements

Your JSON file should follow this structure:

{
  "title": "Your Test Title",
  "description": "Test description",
  "questions": [
    {
      "id": "q1",
      "question": "What is the capital of France?",
      "type": "multiple_choice",
      "options": ["London", "Berlin", "Paris", "Madrid"],
      "correct_answer": "Paris",
      "explanation": "Paris has been the capital since the 12th century.",
      "points": 1,
      "category": "Geography"
    }
  ]
}
Question Types Supported:
  • multiple_choice: Requires 'options' array
  • true_false: Accepts True/False answers
  • fill_blank: Fill in missing words
  • short_answer: Open-ended text answers
Tip: You can download our sample questions file to see the complete format.
